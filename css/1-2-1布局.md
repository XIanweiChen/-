            DOM tree
            The backbone of an HTML document are tags.
            According to Document Object Model (DOM), every HTML-tag is an object. Nested tags are ‚Äúchildren‚Äù of the enclosing one. The text inside a tag it is an object as well.
            All these objects are accessible using JavaScript, we can use them to modify the page.
